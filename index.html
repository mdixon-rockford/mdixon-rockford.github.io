User Defined Objects in JDE

User Defined Objects (UDOs, for short) are Oracle's low-code/no-code solutions that are revolutionizing how companies are using JDE. With each release new features are available that, in many cases, can replace costly, complicated customizations of base JDE software. Instead, UDOs sit above the coding layer and make upgrades or system changes far simpler. And with each release UDOs are providing additional functions that are not easily replicated even with custom software modifications.

Here are just a few simple UDOs we developed to illustrate hwo powerful this tool can be:

Address Book Lookup

Use a web-based service to assist in address book validation

CPI Automation

Use a web-based service to retrieve CPI indices from BLS.gov and populate into JDE

CAFE1 Maps

Link a map to an address in the JDE Address Book automatically
